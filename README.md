# PyMath-Cosmic

[![PyPI version](https://img.shields.io/pypi/v/pymath-cosmic)](https://pypi.org/project/pymath-cosmic/)
[![Python Version](https://img.shields.io/pypi/pyversions/pymath-cosmic)](https://pypi.org/project/pymath-cosmic/)

A simple custom Python math library by Cosmic.

---

## Installation

Works anywhere Python runs (PC, VS Code, Termux, Pydroid 3):

```bash
pip install pymath-cosmic
```
## Usage:

```bash
import pymath

print(pymath.getMathPlus(5, 5))
print(pymath.getMathMinus(5, 2))
print(pymath.getMathMulti(5, 5))
print(pymath.getMathDivide(1, 6))
```
